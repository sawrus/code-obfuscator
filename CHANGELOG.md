# Changelog

## [0.1.0] - 2026-02-22

### Added

- Rust CLI с режимами `forward` и `reverse`.
- Рекурсивная обработка каталогов с сохранением структуры файлов.
- Поддержка ручного `mapping.json`.
- Автогенерация `mapping.generated.json` (forward/reverse карты).
- Опциональная интеграция с Ollama API (`/api/generate`).
- Unit, E2E и SVT blackbox тесты.
- Makefile для сборки, тестов, покрытия и кроссплатформенной сборки.
- Документация проекта в `README.md`.
